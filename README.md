🚀 UniCheck - Hệ thống điểm danh sinh viên Real-time

📌 Giới thiệu

UniCheck là hệ thống điểm danh sinh viên sử dụng công nghệ đám mây, cho phép giảng viên và sinh viên tương tác theo thời gian thực (real-time) mà không cần tải lại trang.

⚙️ Công nghệ sử dụng

- HTML, CSS, JavaScript (thuần)
- Firebase Authentication
- Firebase Firestore (Real-time Database)
- Deploy trên Vercel

---

🔥 Tính năng nổi bật

👨‍🎓 Sinh viên

- Đăng ký / đăng nhập
- Điểm danh bằng QR Code
- Xác thực vị trí GPS
- Xem lịch sử điểm danh

---

👨‍🏫 Giảng viên

- Tạo lớp học
- Mở cổng điểm danh real-time
- Hiển thị danh sách sinh viên ngay lập tức
- Xuất báo cáo Excel (.csv)

---

🛠️ Quản trị viên (Admin)

- Quản lý toàn bộ tài khoản
- Theo dõi Audit Logs theo thời gian thực
- Reset mật khẩu người dùng
- Xóa dữ liệu hệ thống (Format Data)

---

⚡ Tính năng kỹ thuật

- Real-time synchronization (không cần F5)
- Phân quyền 3 cấp (Admin / Teacher / Student)
- Xác thực người dùng bằng Firebase Auth
- Lưu trữ dữ liệu trên Firestore

---

🧠 Kiến trúc hệ thống

Frontend (HTML/CSS/JS)
⬇
Firebase Authentication (Xác thực)
⬇
Firestore Database (Lưu trữ + Real-time)

- QR Code được sử dụng để xác định lớp học
- Một số tính năng yêu cầu cấp quyền camera và vị trí trên trình duyệt
